# Mariana Lungu

I design and build audit-ready AI systems and data architectures for environments where reliability, traceability, and accountability are not optional.

---

## Focus

My work centers on a simple idea:

AI systems are only useful if they can be trusted.

That means designing infrastructure that can answer:

- What data informed this decision?
- Which model and configuration produced it?
- Can the result be reproduced exactly?
- What happens when the system is wrong?
